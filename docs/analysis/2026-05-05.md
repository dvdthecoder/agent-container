## Analysis

- **Qwen2.5-Coder 32B · A100 80GB** — opencode sends **11.9×** more prompt tokens than aider (32,022 vs 2,700). The gap is almost entirely tool-schema overhead: opencode resends all 10 tool schemas (~500 tokens each) on every turn. One of 3 opencode runs failed (run-20260505-072253-d103de): edit tool was confirmed by the proxy but diff was empty — `end_turn` grace period race condition (see [lessons-learned #19](../lessons-learned.md#19-opencode-end_turn-grace-period--file-write-race-condition)).
